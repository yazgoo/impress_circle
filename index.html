Impress circle is a library based on:
  • impress.js
  • impress-arrows
It allows to draw a presentation rotating around a circle.
Here is how you use it.
load the stylesheet
you can add as many normal impress steps as you like
The circle is an html structure in the impress document:
  • Just declare a div of class circle
  • Declare section divs
  • Declare subsections divs
  • Both section and subsection divs use data-name attribute for their title
just include impress_circle.js
include 
impress_circle(doc);
just include impress_arrows_all.js
if you're using arrow:// images, don't forget to run 
impress_arrows().load_images();
